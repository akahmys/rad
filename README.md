# rad (Rust Agent Dispatcher)

`rad` は、知能（ポリシー）と実行（メカニズム）を徹底的に分離した、極小・高速なエージェント指向シェルランタイムです。イベントの分配（Dispatch）と強固なOSプロセス監護を提供します。

`pi-coding-agent` (https://pi.dev/) のミニマリズム思想にインスパイアされて開発を始めました。

---

## 1. 特徴

*   **単一静的バイナリ & ゼロ依存**: Rustによる軽量設計。数MBのメモリフットプリントで超高速起動。
*   **OSレベルのプロセス一括終了（PGID管理）**: コア終了時、エージェントが生成したすべての外部プロセス（孫・ひ孫を含む）をPGID経由で確実に強制終了（`SIGKILL`）。
*   **Gitを汚さない高速ロールバック**: ファイル変更を検知し `.rad/snapshots/` へ自動退避。Git履歴を介さず、任意のチェックポイントへファイルシステムを即座に復元可能。
*   **プラグインの等価性とアクセス制御**: 標準機能とサードパーティ拡張を同一のRPCインターフェースで統合。ケイパビリティマスクにより、不許可なI/Oやコマンド実行を物理的に遮断。

## 2. クイックスタート

### 2.1 前提条件

*   Rust 1.75 以上
*   `cargo`

### 2.2 ビルド手順

```bash
# クローンとビルド
git clone https://github.com/yourusername/rad.git
cd rad
cargo build --release
```

### 2.3 テストの実行

```bash
cargo test
```
